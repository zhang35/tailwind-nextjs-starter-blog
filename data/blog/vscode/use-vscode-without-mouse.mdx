## Editor

Vim mode

## File explorer

Switch between File explorer and editor: `cmd + 0` or `cmd + shift + e`

## Terminal

Open and close terminal: `cmd + j`

Switch between terminal and editor: ``ctrl + ` ``

Switch between multiple terminals: `cmd + shift + ]` and `cmd + shift + [`

## Editor

Split editor: `cmd + \`

Switch between multiple editor tabs: `cmd + 1 / 2 / 3 / 4`
